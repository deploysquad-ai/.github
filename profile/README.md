# deploysquad-ai

> Build AI agents that work.

At [deploysquad.ai](https://deploysquad.ai), we build open-source tools for AI-native development. Run any model — local or remote. Build any agent.

## Projects

- [**squad**](https://github.com/deploysquad-ai/squad) — A model-agnostic multi-agent coding agent. Works with any LLM, local or remote. The coding agent for everyone.
- [**recon**](https://github.com/deploysquad-ai/recon) — AI-powered project knowledge graph for Obsidian. Map your codebase using natural language conversation.

## Contribute

All our projects are open source and we welcome contributions. Open an issue, submit a PR, or start a discussion.

If our tools are useful to you, a ⭐ goes a long way — it helps other developers find us.
